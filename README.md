# PocketPace 🏃‍♂️👝

> **"Your Money, at your own pace."**

PocketPace is a behavioral finance framework designed to eliminate tracking friction and accounting anxiety. Moving completely away from rigid, corporate business ledgers, PocketPace breaks personal money management down into highly intuitive, stress-free **7-day sprints** that adapt to individual lifestyle rhythm.

---

## 🚀 Project Overview

Most personal finance apps fail because they are built on "Accounting Logic"—demanding meticulous data entry, displaying overwhelming multi-month charts, and generating user guilt. 

PocketPace is built on **Behavioral Design Logic**. It treats variable daily spending not as a complex math problem, but as a habit-formation challenge. By centering the entire user experience around a single, highly visual "Weekly Sprint Gauge," it provides real-time clarity right at the cashier counter without the massive cognitive load.

### Core Principles:
* 📉 **Cognitive Load Reduction:** Hides complex calculations behind backend automation, keeping the primary interface incredibly clean.
* 🎛️ **The 4.5-Week Safety Slider:** Replaces decimal heavy divisions with a conservative default engine that automatically generates stress-free weekly allowances and organic surplus rewards.
* 🛡️ **3-Tier Insulation Architecture:** Features a hidden monthly buffer ("shock absorber") to prevent irregular transaction leakage from breaking the user's active weekly motivation streak.

PocketPace is designed from the ground up by a Creative Technologist to cleanly bridge the gap between human psychology and frontend engineering.
